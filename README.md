# 📝 Task Mate

Task Mate is a simple React app to add, manage, and track your daily tasks easily.

## 🚀 Live Demo

👉 [Check out the live demo here!](https://taskmate-kashaboina.netlify.app/)

## 📌 Features

- Add new tasks
- Mark tasks as complete
- Delete tasks
- Simple and clean UI

## ⚙️ Tech Stack

- React
- HTML
- Css
